
# 🏦 Ibank - Digital Banking System

Ibank is a modern banking application designed to simulate real-world financial operations. The project demonstrates how core banking systems work, including account management, transactions, and secure handling of funds.

## 🚀 Features

- User account creation and management
- Deposit and withdrawal system
- Secure money transfers between accounts
- Transaction history tracking
- Balance management
- Basic authentication and validation
